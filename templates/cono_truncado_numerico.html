1. Estructura General del Método

Implementamos exactamente la misma estructura vista en clase:

def h_num_cono_truncado(h_0, paso, r, R0, H, a, g):
    aux = int(h_0 / paso)
    t_num = []
    y_num = [] 
    error_num = []
    y = h_0
    
    for i in range(1, aux + 1):
        value, error = integrate.quad(integrando, y, h_0)
        y_num.append(y)
        t_num.append(value) 
        error_num.append(error)
        y = y - paso
        
    return t_num, y_num, error_num
2. Función Integrando para Cono Truncado

Para el cono truncado, el integrando es:

\( \displaystyle \frac{dt}{dh} = \frac{A(h)}{a\sqrt{2gh}} \)
\( \displaystyle A(h) = \pi \left[r + \frac{R_0 - r}{H}h\right]^2 \)

En código:

lambda h: A_h(h, r, R0, H) / (a * np.sqrt(2 * g * h))
3. Límites de Integración

Integramos desde la altura actual \(y\) hasta la altura inicial \(h_0\):

\( \displaystyle t(h) = \int_{h}^{h_0} \frac{A(\xi)}{a\sqrt{2g\xi}} d\xi \)
integrate.quad(integrando, y, h_0)
4. Malla Descendente en Altura

Generamos puntos desde \(h_0\) hasta 0 con el paso especificado:

\( h = h_0, h_0 - \text{paso}, h_0 - 2\cdot\text{paso}, \dots \)
y = y - paso en cada iteración
5. Resultados y Errores

El método retorna tres listas como en clase:

t_num: Tiempos calculados para cada altura
y_num: Alturas de la malla descendente
error_num: Errores de integración estimados por quad
