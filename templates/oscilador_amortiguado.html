2.1 Oscilador amortiguado (según la guía)

En la guía se parte del sistema general: \[ \frac{d^2x}{dt^2} + \Gamma \frac{dx}{dt} + \omega_0^2 x = \frac{F_0}{m}\cos(\omega_f t), \] con \(\omega_0^2 = k/m\), \(\Gamma = b/m\).

Para el oscilador amortiguado se toma \(F_{\text{ext}}=0\): \[ \frac{d^2x}{dt^2} + \Gamma \frac{dx}{dt} + \omega_0^2 x = 0. \]

Sistema de ecuaciones (como en el código):

Definimos \(x = x(t)\), \(v = x'(t)\). Entonces:

\[ \begin{cases} \dfrac{dx}{dt} = v,\\[4pt] \dfrac{dv}{dt} = -\Gamma v - \omega_0^2 x. \end{cases} \] 

def modelo_3(z, t):
    x = z[0]
    v = z[1]
    dzdt = [0.0, 0.0]
    dzdt[0] = v
    dzdt[1] = - Gamma * v - omega_0**2 * x
    return dzdt

Abajo a la derecha verás x(t) y v(t) como en el cuaderno: mismos parámetros iniciales, pero ahora con el bloque y un punto que se mueven con el tiempo.

Usa primero los valores por defecto (los del profe), observa el amortiguamiento, luego cambia \(b\), \(k\), \(m\), \(x_0\), \(v_0\) y ve cómo cambian las curvas y el movimiento.
